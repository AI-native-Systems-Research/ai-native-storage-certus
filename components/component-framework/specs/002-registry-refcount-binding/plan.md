## Summary

Add a component registry with factory pattern, explicit reference counting via ComponentRef (wrapping Arc), and third-party binding (string-based interface/receptacle name matching with TypeId verification). Extends the existing component-core crate with new types and adds a `connect_receptacle_raw` method to IUnknown for runtime wiring.

## Design Decisions

### ComponentRef wraps Arc<dyn IUnknown>
- `attach()` = `Arc::clone()`, `release()` = `drop()`
- Implements `Deref<Target = dyn IUnknown>` for transparent access
- Implements `Clone` (same as attach)
- `From<Arc<T>>` where `T: IUnknown` for ergonomic construction

### ComponentRegistry uses RwLock<HashMap>
- Concurrent reads, exclusive writes
- `register()` rejects duplicates, `create()` invokes factory
- Factory panics caught via `std::panic::catch_unwind`

### Third-party binding via connect_receptacle_raw
- New IUnknown method generated by macro
- Match arms for each receptacle: downcast `Box<dyn Any>` to `Arc<dyn IFoo>`
- `bind()` free function orchestrates: find interface by name → query → connect

### Factory accepts Option<&dyn Any> config
- Type-erased, optional configuration
- Each factory downcasts to its expected config type
- Uniform signature across all factories
