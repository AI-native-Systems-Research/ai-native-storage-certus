# Data Model: COM-Style Component Framework

**Date**: 2026-03-30
**Feature**: 001-com-component-framework

## Core Types

### Interface (trait)

Every user-defined interface is a Rust trait with the following
constraints enforced by the `define_interface!` macro:

- Extends `Send + Sync + 'static`
- Method signatures may include lifetime parameters on arguments/returns
- All methods take `&self` (no `&mut self`; mutation via interior
  mutability in implementations)

**Identity**: `TypeId::of::<dyn IFoo + Send + Sync>()` uniquely
identifies the interface at runtime.

### InterfaceInfo

Lightweight metadata for runtime introspection.

| Field       | Type            | Description                              |
|-------------|-----------------|------------------------------------------|
| `type_id`   | `TypeId`        | Unique identity of the interface         |
| `name`      | `&'static str`  | Human-readable interface name            |

### ReceptacleInfo

Metadata for a required interface slot.

| Field            | Type            | Description                         |
|------------------|-----------------|-------------------------------------|
| `type_id`        | `TypeId`        | TypeId of the required interface    |
| `name`           | `&'static str`  | Human-readable receptacle name      |
| `interface_name` | `&'static str`  | Name of the required interface type |

### Receptacle\<T\>

A typed slot representing a required interface.

| Field        | Type                                         | Description                |
|--------------|----------------------------------------------|----------------------------|
| `connection` | `RwLock<Option<Arc<dyn T + Send + Sync>>>`   | Current provider, if any   |

**State transitions**:

```text
Disconnected ──connect(provider)──▶ Connected
     ▲                                   │
     └────────disconnect()───────────────┘

Disconnected + invoke() → Err(NotConnected)
Connected + connect()   → Err(AlreadyConnected)
Connected + invoke()    → delegates to provider
Connected + disconnect()→ Disconnected (drops Arc ref)
```

### IUnknown (trait)

Base interface for all components. Object-safe via raw methods;
typed wrappers generated by macro.

| Method                    | Signature                                                    | Description                              |
|---------------------------|--------------------------------------------------------------|------------------------------------------|
| `query_interface_raw`     | `(&self, TypeId) -> Option<&(dyn Any + Send + Sync)>`        | Type-erased interface lookup             |
| `version`                 | `(&self) -> &str`                                            | Component version string                 |
| `provided_interfaces`     | `(&self) -> &[InterfaceInfo]`                                | List of all provided interfaces          |
| `receptacles`             | `(&self) -> &[ReceptacleInfo]`                               | List of all required interface slots     |

### InterfaceMap

Internal storage for a component's provided interfaces.

| Field      | Type                                          | Description                      |
|------------|-----------------------------------------------|----------------------------------|
| `map`      | `HashMap<TypeId, Box<dyn Any + Send + Sync>>` | TypeId → boxed `Arc<dyn IFoo>`   |
| `info`     | `Vec<InterfaceInfo>`                          | Ordered list for enumeration     |

Populated once at component construction. Immutable thereafter.

### Component (generated struct)

Each `define_component!` invocation generates a struct with:

| Field             | Type                    | Description                        |
|-------------------|-------------------------|------------------------------------|
| `interfaces`      | `InterfaceMap`          | Provided interfaces                |
| `receptacle_info` | `Vec<ReceptacleInfo>`   | Metadata for required interfaces   |
| `version`         | `&'static str`          | Component version                  |
| (user fields)     | (user-defined)          | Domain-specific state              |
| (receptacles)     | `Receptacle<dyn IFoo>`  | One per declared receptacle        |

### Error Types

| Variant              | Context                                    |
|----------------------|--------------------------------------------|
| `NotConnected`       | Receptacle invoked while disconnected      |
| `AlreadyConnected`   | Connect called on already-connected slot   |
| `InterfaceNotFound`  | query_interface for unsupported interface  |

## Relationships

```text
Component ──implements──▶ IUnknown
Component ──provides────▶ 1..N Interface (via InterfaceMap)
Component ──requires────▶ 0..N Receptacle<dyn IFoo>
Receptacle ──connects──▶ 0..1 Arc<dyn IFoo> (provider)
InterfaceMap ──contains──▶ InterfaceInfo[]
Component ──contains──▶ ReceptacleInfo[]
```

## Invariants

1. Every component provides at least `IUnknown` (enforced by macro).
2. `InterfaceMap` is immutable after construction (no dynamic
   interface addition/removal).
3. A `Receptacle` holds at most one connection at any time.
4. All types are `Send + Sync`.
5. `Arc`-based sharing prevents use-after-free: a provider component
   stays alive as long as any receptacle references it.
