# Data Model: SPDK CPU Memory Allocator Component

**Date**: 2026-04-10
**Feature**: 001-spdk-memory-alloc

## Entities

### DmaBuffer (external — defined in `interfaces` crate)

A DMA-safe memory buffer handle. Used as the return type for all allocation operations and consumed by free/reallocate.

| Field | Type | Description |
|-------|------|-------------|
| ptr | `*mut c_void` | Raw pointer to hugepage-backed memory |
| len | `usize` | Buffer length in bytes |
| free_fn | `unsafe extern "C" fn(*mut c_void)` | Deallocator function (spdk_dma_free or spdk_free) |
| numa_node | `i32` | NUMA node ID (-1 = unknown/any) |
| metadata | `BTreeMap<String, String>` | Optional key-value metadata |

**Lifecycle**: Created by allocate/zmalloc → used by caller → consumed by free/reallocate → Drop calls free_fn.

### AllocationStats (new — defined in `interfaces` crate, `spdk_types.rs`)

Immutable snapshot of current memory allocation state. Returned by the `stats()` method.

| Field | Type | Description |
|-------|------|-------------|
| total_bytes_allocated | `u64` | Sum of all active allocation sizes across all zones |
| total_allocation_count | `u64` | Number of active allocations across all zones |
| zones | `HashMap<i32, ZoneStats>` | Per-NUMA-zone breakdown, keyed by NUMA node ID |

**Identity**: Value type (Clone, Debug, PartialEq). No unique identifier — each call to `stats()` produces a fresh snapshot.

**Validation**: `total_bytes_allocated == zones.values().map(|z| z.bytes_allocated).sum()` and same for count.

### ZoneStats (new — defined in `interfaces` crate, `spdk_types.rs`)

Per-NUMA-zone allocation statistics.

| Field | Type | Description |
|-------|------|-------------|
| bytes_allocated | `u64` | Total bytes currently allocated in this zone |
| allocation_count | `u64` | Number of active allocations in this zone |

**Identity**: Value type (Clone, Debug, PartialEq, Default).

### MemoryAllocatorError (new — defined in `interfaces` crate, `spdk_types.rs`)

Error type for memory management operations.

| Variant | Payload | Description |
|---------|---------|-------------|
| EnvNotConnected | `String` | ISPDKEnv receptacle not connected |
| EnvNotInitialized | `String` | SPDK environment not initialized |
| InvalidSize | `String` | Size parameter is zero |
| InvalidAlignment | `String` | Alignment parameter is zero |
| AllocationFailed | `String` | SPDK returned NULL (memory exhausted) |
| ReallocFailed | `String` | Reallocation failed (new alloc failed) |

**Traits**: `Debug, Clone, Display, Error`

### StatsInner (internal — defined in `memory-allocator-cpu` crate)

Mutable internal stats state, protected by Mutex. Not exposed publicly.

| Field | Type | Description |
|-------|------|-------------|
| total_bytes | `u64` | Running total of allocated bytes |
| total_count | `u64` | Running count of active allocations |
| zones | `HashMap<i32, ZoneStats>` | Mutable zone map |

**Thread safety**: Protected by `Mutex<StatsInner>`. Lock held only during counter updates.

### MemoryAllocatorCpu (component — defined in `memory-allocator-cpu` crate)

The component struct, generated by `define_component!`.

| Field | Type | Description |
|-------|------|-------------|
| spdk_env | `Receptacle<dyn ISPDKEnv + Send + Sync>` | Auto-generated receptacle for SPDK environment |
| stats | `Mutex<StatsInner>` | Thread-safe allocation statistics |

**Lifecycle**: Created via `MemoryAllocatorCpu::new(...)` → receptacle wired → allocate/free/zmalloc/reallocate/stats called → Drop (no special cleanup needed beyond stats).

## Relationships

```text
MemoryAllocatorCpu ──receptacle──▶ ISPDKEnv (provided by SPDKEnvComponent)
         │
         │ implements
         ▼
  IMemoryManagement
         │
         │ returns/consumes
         ▼
     DmaBuffer ◄── backed by SPDK hugepage memory
         │
         │ tracked in
         ▼
   AllocationStats ──contains──▶ ZoneStats (per NUMA node)
```

## State Transitions

**Stats state** (per allocation lifecycle):

```text
[No allocation]
    │
    │ allocate/zmalloc succeeds
    ▼
[Active allocation]
  stats: +size bytes, +1 count in zone[numa_node]
    │
    ├── free() called
    │   stats: -size bytes, -1 count in zone[numa_node]
    │   → [No allocation]
    │
    └── reallocate() called
        stats: -old_size, -1 in zone[old_numa]
               +new_size, +1 in zone[new_numa]
        → [Active allocation] (new buffer)
```
